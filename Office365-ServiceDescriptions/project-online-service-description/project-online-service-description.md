# <a name="project-online-service-description"></a>Description du service Project Online

Project Online est une solution flexible en ligne pour la gestion de portefeuille de projets (PPM) et des tâches quotidiennes. Remis via Office 365, Project Online fournit des fonctionnalités de gestion de projet puissants pour la planification, l’ordre de priorité et la gestion des projets et des investissements de portefeuille de projet — depuis pratiquement n’importe où sur quasiment n’importe quel appareil. Project Online peut être utilisé par les administrateurs, responsables de portefeuilles visionneuses, responsables de projets et des ressources et responsables d’équipe et des membres. 
  
Project Online est disponible dans le cadre de trois offres, Project Online Essentials, Project Online Professionnel et Project Online Premium.
  
Pour comparer les fonctionnalités de chacune des offres, reportez-vous au [comparatif des offres Project Online](https://products.office.com/en-us/project/compare-microsoft-project-management-software).
  
Vous pouvez exporter, enregistrer et imprimer des pages dans les descriptions du service Office 365. Découvrez comment [exporter plusieurs pages](https://go.microsoft.com/fwlink/?LinkId=403349).
  
## <a name="project-online-boundaries-and-limits"></a>Limites et restrictions de Project Online

Il existe certaines limites au service Project Online. Pour plus d'informations, consultez la rubrique [Project Online : limites et frontières logicielles](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489).
  
## <a name="data-backup"></a>Sauvegarde de données

 Project Online repose sur SharePoint Online et suit la même stratégie de sauvegarde de données que SharePoint. Pour plus d'informations, consultez la section Sauvegarde de données dans [Description du service SharePoint Online](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx).
  
## <a name="use-of-sharepoint-online"></a>Utilisation de SharePoint Online

Votre service Project Online requiert l’utilisation de SharePoint Online, qui sera déployé pour vous dans le cadre du service Project Online. Droits pour les fonctionnalités de SharePoint Online fournies avec Project Online Professional ou Premium sont limitées pour le stockage et l’accès aux données pour prendre en charge de Project Online.
  
## <a name="licensing-considerations"></a>Remarques sur les licences

### <a name="internal-users"></a>Utilisateurs internes

Aucune intervention sur un site Project Online nécessite une licence Project Online.
  
- Project Online Essentials est la configuration minimale requise.
    
- Pour les cas d’utilisation plus avancées, les utilisateurs doivent disposer de licences appropriés. Responsables de projets nécessitent au moins une licence de Project Professional en ligne. Responsables de portefeuilles nécessitent une licence de Project Online Premium.
    
> [!IMPORTANT]
>  Project Online Essentials est *une licence pour les clients qui ont Project Online Professional ou Project Online Premium membre d’équipe* .<br/>Lorsque votre licence Project Online Professional ou Project Online Premium dernière arrive à expiration, les instances de Project Web App sont supprimés après 120 jours.<br/>Pour les abonnements d’évaluation Project Online, votre instance Project Web App sera supprimé 30 jours après la fin de la période d’évaluation. 
  
### <a name="external-users"></a>Utilisateurs externes

Gestion des licences pour les utilisateurs externes de Project Online suit la stratégie de même que pour les utilisateurs internes. Aucune intervention sur un site Project Online nécessite une licence Project Online.
